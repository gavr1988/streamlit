# Global Movers Driver's Portal

This is a small Streamlit app for the Global Movers Driver's Portal.

## Features

- Dashboard view with company guidance and operating notes
- Weight converter for `kg <-> lbs`

## Requirements

- Python 3.9 or newer
- Streamlit

## Setup

Install the dependency:

```bash
pip install streamlit
```

## Run the app

Start the app from the project folder:

```bash
streamlit run streamlit.py
```

## Project Structure

- `streamlit.py` - main Streamlit application
- `README.md` - project overview and run instructions

## Notes

- The app uses a sidebar to switch between the dashboard and the converter tool.
- If you use a virtual environment, keep it outside version control.
